# 开发问题汇总

标志用于区分问题种类，加*号表示 待讨论

- F -----   FileSystem 文件系统
- P -----  Progress 进程 ，限制在进程正常控制流内的工作，包括父子进程
- M -----  Memory 内存管理
- *T ----- trap 中断和异常， 包括系统调用，进程切换，硬件定时器，软件定时器等
- B ----- bug
- *Sch----- 调度Schedule 
- *Syn ---- 同步
- ....
